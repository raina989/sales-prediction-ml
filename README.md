## Project Overview
This project uses machine learning to predict sales based on historical transaction data. The goal is to understand how basic features such as quantity, discount, and seasonality influence sales performance.

## Objective

- Evaluate model performance using standard regression metrics
- Visualize actual vs predicted sales values

## Dataset

## Features Used

- Discount
- Month

## Model Used
